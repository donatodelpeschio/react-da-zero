## 🚀 Come usarlo

1. Clona il repo o scarica `corso-react.html`
2. Aprilo nel browser (Chrome, Firefox, Safari — qualsiasi va bene)
3. Studia una lezione alla volta usando la navigazione laterale

Oppure, accedi direttamente al 
corso [cliccando qui](https://donatodelpeschio.github.io/react-da-zero/).

## 📚 Contenuti

### Modulo 1 — Fondamentali
- Cos'è React (Virtual DOM, JSX, setup con Vite)
- Il tuo primo componente
- Props e composizione
- State con `useState`
- Rendering condizionale e liste

### Modulo 2 — Hook Essenziali
- `useEffect` — effetti collaterali e fetch dati
- `useRef` — accesso al DOM e valori persistenti
- `useContext` — stato condiviso senza prop drilling
- Custom Hooks — logica riutilizzabile

### Modulo 3 — Gestione Dati
- Fetch e dati asincroni (pattern loading/errore/dati)
- Lifting state up

### Modulo 4 — Routing
- React Router v6 (route, link, parametri, layout)

### Modulo 5 — Stato Globale
- `useReducer`
- Zustand

### Modulo 6 — Performance e Architettura
- `useMemo` e `useCallback`
- Lazy loading e code splitting con Suspense
- Struttura di un progetto reale
